# Basic Terms Inside Head Tag
```


<!-- <!DOCTYPE html>   -  it tells search engine the html version use  -->
<!DOCTYPE html>

<!-- dir = ltr   - write text from left to right  -->
<html lang="en" dir="ltr">
  <head>

    <!-- meta charset utf-8  -  it provide developer to access all symbols and characters in the world -->
    <meta charset="utf-8">

    <!-- name = keywords - this attribute tells search engine the main keywords for search this web page -->
    <meta name="keywords" content="web development , html">

    <!-- name = author - this attribute tells search engine the author/developer name of the web page -->
    <meta name="author" content="Prince Sharma">

    <!-- name = description  - on browser some text under the website link that describe the webpage content -->
    <meta name="description" content="source code of this web page gives information about the meta tag and its various attribute">

    <!-- http-equiv - it tells search engine to refresh the webpage in every 10 seconds -->
    <meta http-equiv="refresh" content="10">


    <!--  title - title of webpage -->
    <title> Meta Tag and its Attributes </title>


  </head>
  <body>


  </body>
</html>

```
